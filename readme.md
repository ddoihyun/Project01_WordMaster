# **# Project1: WordMaster**

### < Class 구성 목록 >

- ICRUD : CRUD를 위한 interface
- Word : 데이터클래스
- WordCRUD : ICRUD interface 구현제
- WordManager : WordCRUD를 사용한 데이터 처리
- Main : static main 함수, starter class
